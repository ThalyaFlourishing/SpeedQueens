SPEED QUEENS
FUNCTION NAME FUNCTION DESCRIPTION
permTestEngine Test engine: Generates permutation arrays, sends them to the tester, and displays test results
nQueensTestPositional Diagonal-collision tester: Tests by computing position numbers and testing those numbers
nQueensTestIndexee Diagonal-collision tester: Tests by comparing differences between array indices and their values
permAnimate The top-level display function; takes n and sequentially displays all permutations of n
permGenArray Generates an array called 'window.permutationsArray' of all permutations of 1 through n.
swap Helper function for permutation engine: swaps two numbers (in-place of given positions in an array
starterArrayGen Creates and returns a single array of length n
translatePermutationToNumericMatrix Takes a permutation array and returns a character matrix of '1's and '0's
translatePermutationToCharacterMatrix Takes a permutation array and returns a character matrix of queens and blanks
displayPermutationMatrix Displays a single permutation array as a chess board
showBoards Used by permAnimate to cycle through all elements of a permutations array and call 'showBoard' to display each one
showBoard Takes a single permutation and displays it as a chessboard
analysis Calculates and reports a comparison of the number of possible permutations of n versus the number of possible arrangements of n on an n^2 board
factorial Helper function for 'analysis' which returns the factorial value of n
isUnique Duplication-tester for nQueensTestIndexee. Returns boolean re: whether the given array has any duplicate values
